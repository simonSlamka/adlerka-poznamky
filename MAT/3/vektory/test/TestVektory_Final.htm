VEKTORY

  
Z daných možností vyberte práve jednu správnu odpoveď.
Odpovedzte SPRÁVNE na všetky otázky, máte na to 30 minút.
(Pozn.: vektor u⃗ znamená vektor so šípkou nad u.)

 
  1. Dané sú body A[-1; 1] a B[2; -7]. Vektor u⃗ = AB⃗ =
    1.   (3; -8)
    2.   (1; -8)
    3.   (-3; -6)
    4.   (3; 8)
  2. Vrcholy trojuholníka PQR tvoria body P[1; 4], Q[-5; 0] a R[-3; -2]. Veľkosť ťažnice na stranu "p" je
    1.   √17
    2.   5√2
    3.   2√17
    4.   √2
  3. Vektor a⃗ = (1; x) je jednotkový. Súradnica x =
    1.   0
    2.   1
    3.   -1
    4.   ±1
  4. Dané sú vektory u⃗ = (-2; 3) a v⃗ = (4; -1). Vektor w⃗ = 10∙u⃗ - 2∙v⃗ =
    1.   (-28; 32)
    2.   (24; 32)
    3.   (-28; 28)
    4.   (24; 28)
  5. Dané sú dva lineárne závislé vektory u⃗ = (-12; -4) a v⃗ = (x; -2). Súradnica x =
    1.   -6
    2.   6
    3.   0,5
    4.   -0,5
  6. Vektor c⃗ = (2; 10) je lineárnou kombináciou vektorov a⃗ = (1; 3) a b⃗ = (-2; 2). Vektor c⃗ =
    1.   3a⃗ + 0,5b⃗
    2.   4a⃗ + a⃗
    3.   6a⃗ + 2b⃗
    4.   -2a⃗ - 2b⃗
  7. Vektor v⃗ je rovnobežný s vektorom u⃗ = (4; -3) a spĺňa podmienku u⃗ ∙ v⃗ = -50. Vektor v⃗ =
    1.   (-8; 6)
    2.   (8; -6)
    3.   (-5; 10)
    4.   (5; -10)
  8. Skalárny súčin vektorov u⃗ = (4; -7) a v⃗ = (2; -3) je
    1.   -13
    2.   -26
    3.   29
    4.   -16
  9. Vrcholy trojuholníka KLM tvoria body K[3; -2], L[4; 1] a M[2; 0]. Veľkosť jeho vnútorného uhla pri vrchole L je
    1.   45°
    2.   90°
    3.   30°
    4.   135°
  10. Vektory u⃗ = (-4; -6) a v⃗ = (x; 8) sú na seba kolmé. Súradnica x =
    1.   -10
    2.   10
    3.   12
    4.   -12

# Komplexné čísla

- Množina komplexných čísle - C

- Zavedenie (Leo Euler) čísla **i**(imaginárna jednotka) - i = sqrt(-1)

### Príklad

Vypočítajte mocniny: i^3, i^3, i^5, (-i)^2, i^17, i^100

i^3 = i * i^2 = -i

i^4 = (i^2)^2 = 1

i^5 = (i^4) * i = i

(-i)^2 = i^2 = -1

i^17 = (i^2)^8 * i = i

i^100 = (i^2)^50 = 1

Riešte v C

x^2 + 1 = 0
